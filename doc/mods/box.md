@use box

# box Module

Allows to display text in boxes with different flavors. To use it:

```
@use box
```
And we can get:

<box info>
This is an info!
</box>

<box alert>
This is an alert!
</box>

<box download>
This is a download!
</box>

<box help>
This is help!
</box>

<box important>
This is important!
</box>

<box tip>
This is a tip!
</box>

<box todo>
This is a todo!
</box>

<box todo rounded>
This is a rounded todo!
</box>

<box color=red text=blue rounded|Custom>
Custom!
</box>
